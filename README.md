# 🧺 Laundry Booking System

A responsive laundry booking web application built using **HTML, CSS, and JavaScript**. The project demonstrates **DOM Manipulation** by allowing users to browse laundry services, add them to a cart, calculate the total amount, and submit a booking form.


## 🚀 Features

- Browse multiple laundry services
- Add services to the cart
- Skip to the next available service
- Dynamic cart updates
- Automatic total price calculation
- Booking form with validation
- Responsive and clean user interface


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Flexbox

## ⚙️ How It Works

1. Browse the available laundry services.
2. Click **Add Item** to add the current service to the cart.
3. Click **Skip Item** to view the next service.
4. The cart updates dynamically with:
   - Serial Number
   - Service Name
   - Price
5. The total amount is calculated automatically.
6. Fill in the booking form and click **Book Now**.
